\section{Signal Chain \& FPGA/AWG}
\label{sec:signal-chain}

This section covers the complete MW generation and detection signal chain for D0/D1
ODMR operation, the FPGA Direct Digital Synthesis (DDS) design, and the hardware
decision between the Artix-7 stack and the AMD RFSoC 4x2 platform.

\subsection{D0 Signal Chain (CW-ODMR)}

For D0 (continuous-wave ODMR), the signal chain is:
\begin{enumerate}
  \item \textbf{ADF4351 MW synthesizer}: stepped from 2800--2950\,MHz at 0.1\,MHz increments;
    output $+5$\,dBm~\cite{ADF4351datasheet}. % ADF4351datasheet p.4 Table 1
  \item \textbf{ADL5321 RF driver amplifier}: gain 11.7\,dB at 2877\,MHz;
    $P_{1\text{dB}} = 25.7$\,dBm~\cite{ADL5321datasheet}.
    % ADL5321datasheet Table 2 (S21 interpolated: 11.71 dB at 2850 MHz); Table 1 (P1dB at 3.5 GHz)
  \item \textbf{MW delivery to diamond}: via loop antenna or PCB stripline near NV ensemble.
  \item \textbf{NV fluorescence detection}: confocal objective $\to$ dichroic $\to$ photodetector
    (APD or SiPM, depending on milestone).
  \item \textbf{Lock-in / software averaging}: ODMR contrast read vs.\ frequency step.
\end{enumerate}

\subsection{D1 Signal Chain (Pulsed ODMR)}

For D1 (pulsed ODMR / Ramsey magnetometry), the MW chain adds the ADL5375 IQ modulator
for phase control and pulsed-envelope shaping~\cite{ADL5375datasheet}:
% ADL5375datasheet (IQ modulator, D1 upgrade path, not needed for D0; PDF in hand)
\begin{equation*}
  \text{ADF4351 LO}
  \;\to\;
  \text{ADL5375 IQ mod.}
  \;\leftarrow\;
  \text{FPGA DDS baseband}
  \;\to\;
  \text{ADL5321 PA}
  \;\to\;
  \text{NV diamond}.
\end{equation*}
The FPGA generates the baseband I/Q signals (0, $\pm 2.162$\,MHz for the NVQbit tripod
protocol, or simple pulsed envelope for Ramsey) via on-chip DDS.

\subsection{FPGA DDS Architecture}

The Artix-7 XC7A100T contains 240 DSP48E1 blocks, each a 25$\times$18-bit
multiplier-accumulator capable of running at up to 464\,MHz~\cite{XilinxUG479}.
% XilinxUG479 p.47 (max clock from DS181 Artix-7 data sheet referenced therein)
A complete DDS core for one tone requires $\sim$1--2 DSP48E1 blocks (32-bit accumulator +
LUT); three tones require $\sim$6 DSP48E1 blocks.
