\section{SiPM Readout Design}
\label{sec:sipm-readout}

The D1 photon-counting readout chain detects NV fluorescence using a silicon photomultiplier
(SiPM) coupled to an FPGA-based timing and control system. The selected architecture
(Option~A) replaces multiple standalone instruments with a single-board FPGA solution.
% internal: hardware/sipm_fpga/ARCHITECTURE_SUMMARY.tex (architecture decision 2026-03-18)

\subsection{Selected Architecture: Option A (SiPM + Fast Comparator + FPGA)}

The readout chain is:
\begin{equation*}
  \underbrace{\text{S14420 SiPM}}_{\substack{\text{Hamamatsu}\\\text{Vis/NIR MPPC}}}
  \;\to\;
  \underbrace{\text{MAX40658 TIA}}_{\substack{18.3\,\mathrm{k}\Omega\\\text{transimpedance}}}
  \;\to\;
  \underbrace{\text{ADCMP601 comp.}}_{\substack{3.5\,\mathrm{ns}\\\text{prop. delay}}}
  \;\to\;
  \underbrace{\text{Nexys A7 FPGA}}_{\substack{\text{XC7A100T}\\\text{TDC + control}}}
  \;\to\; \text{PC}.
\end{equation*}

\subsection{Why Option A}

\begin{enumerate}
  \item \textbf{Minimal analog complexity:} 2 ICs (TIA + comparator) on a hand-solderable
    2-layer PCB.
  \item \textbf{SiPM wavelength match:} The S14420 is engineered for visible/NIR detection
    (637--800\,nm NV emission band~\cite{Doherty2013}), % Doherty2013 Fig.~1(b) p.5 (ZPL at 637 nm, phonon sideband)
    with enhanced red sensitivity vs.\ standard NUV-optimized SiPMs~\cite{S14420datasheet}.
    % S14420datasheet p.3 Fig. (PDE vs. wavelength, 50 um cell curve shows peak at 550-600 nm)
  \item \textbf{FPGA TDC resolution:} Artix-7 CARRY4 carry-chain TDC achieves
    17.5\,ps resolution~\cite{Lim2025}, % Lim2025 Table 4 p.13 (4-tap SCSC LSB = 17.530 ps)
    far exceeding the $\mu$s-scale ODMR dwell time (20\,ms per step, $+5$\,dBm,
    0.1\,MHz steps; This work) and enabling future $T_1$ lifetime measurements.
  \item \textbf{Cost:} Preliminary all-in estimate $\approx\!\$500$--600 for the full
    readout chain (SiPM + TIA PCB + comparator + FPGA board + cables).
    \needref{Formal vendor quote for complete BOM.}
  \item \textbf{FPGA replaces instruments:} A single Nexys A7 board simultaneously
    implements the pulse sequencer, MW SPI controller, photon counter/TDC, histogram
    accumulator, and PC interface — replacing multiple standalone instruments~\cite{NexysA7refmanual}.
    % NexysA7refmanual p.1 (board overview, PMOD connectors for instrument interfacing)
\end{enumerate}

\subsection{FPGA Functions}

The Nexys A7-100T FPGA (XC7A100T) implements the following subsystems in HDL:
\begin{enumerate}
  \item \textbf{ADF4351 SPI controller (\texttt{spi\_master/adf4351\_ctrl}):}
    3-wire SPI, 32-bit MSB-first, CLK $\leq$20\,MHz, LE $\geq$20\,ns, R5$\to$R0
    initialization order~\cite{ADF4351datasheet}. % ADF4351datasheet p.18-22 (SPI timing Table 2, register write sequence)
    Step size 0.1\,MHz; sweep rate set by PC (This work).
  \item \textbf{Pulse sequencer (\texttt{pulse\_sequencer}):}
    State machine controlling laser AOM gate and MW enable. Timing resolution:
    10\,ns at 100\,MHz system clock (This work).
  \item \textbf{Carry-chain TDC (\texttt{tdc\_core}):} 17.5\,ps-resolution photon
    timestamping using CARRY4 chains~\cite{Lim2025,Wu2010WaveUnion}.
    % Lim2025 Table 4 p.13 (17.530 ps resolution on Artix-7 CARRY4 TDL); Wu2010WaveUnion (Wave Union method)
  \item \textbf{Photon counter (\texttt{photon\_counter}):}
    Gated 32-bit counter for CW-ODMR mode.
  \item \textbf{Histogram accumulator (\texttt{histogram\_bram}):}
    2048-bin photon arrival-time histogram in Block RAM (one bin per ODMR frequency
    step at 0.1\,MHz resolution = 204.8\,MHz sweep range). Accumulated over the
    dwell time (20\,ms per step; This work).
    % NexysA7refmanual p.1 Table (4,860 Kb BRAM — adequate for 2048-bin histogram)
  \item \textbf{UART interface (\texttt{uart\_interface}):}
    Histogram data transmitted to PC at end of sweep (This work).
\end{enumerate}

\subsubsection{Stage 1 — CW Carrier: ADF4351 Module}
\label{sec:adf4351}

The ADF4351 breakout module provides the CW MW carrier at 2877.5\,MHz (NV zero-field
splitting at cryogenic temperature; This work). Key register values for CW-ODMR:

\begin{itemize}
  \item $f_\mathrm{ref} = 25$\,MHz (TCXO on breakout module — verify before programming).
  \item MOD $= 250$ for 0.1\,MHz step resolution~\cite{ADF4351datasheet}.
    % ADF4351datasheet p.15 Eq. (1): f_out = f_PFD * (INT + FRAC/MOD)
  \item Output power: $+5$\,dBm (Register~4, DB4:DB3 = 11, four discrete levels
    $-4$, $-1$, $+2$, $+5$\,dBm in 3\,dB steps~\cite{ADF4351datasheet}).
    % ADF4351datasheet p.20 Table Register 4 (RF output power bits DB4:DB3)
  \item Phase noise floor: $-148$\,dBc/Hz at 5\,MHz offset from 3.3\,GHz carrier~\cite{ADF4351datasheet}.
    % ADF4351datasheet p.6 Table (typical phase noise specifications)
\end{itemize}

\subsubsection{Stage 2 — IQ Modulator Upgrade Path (D1 Pulsed / D2+)}
\label{sec:iq-upgrade}

For pulsed ODMR requiring nanosecond phase switching (Ramsey, Hahn echo, $T_2$
measurements), the ADF4351 PLL relock latency ($\sim$5--15\,$\mu$s) is inadequate.
The upgrade inserts an ADL5375 IQ modulator between the ADF4351 and the NV diamond:
\[
  \underbrace{\text{ADF4351}}_{\substack{\text{CW LO}\\\text{2877.5\,MHz}}}
  \;\to\;
  \underbrace{\text{ADL5375}}_{\substack{\text{IQ mod}\\\text{400\,MHz--6\,GHz}}}
  \;\leftarrow\;
  \underbrace{\text{FPGA}}_{\text{digital I/Q}}
  \;\to\;
  \underbrace{\text{ADL5321}}_{\substack{\text{2.3--4.0\,GHz}\\\text{amp}}}
  \;\to\; \text{NV.}
\]

The ADL5375 covers 400\,MHz--6\,GHz and is fully functional at 2877\,MHz~\cite{ADL5375datasheet}.
% ADL5375datasheet p.3 Table (frequency range)
Phase switching time is FPGA GPIO slew-limited at $\sim$2--5\,ns, enabling the
minimum Ramsey free-precession window $\tau_\mathrm{min} \approx 5$\,ns —
well below NV $T_2^*$ at any operating temperature (This work).
The ADL5321 provides 11.7\,dB gain at 2877\,MHz~\cite{ADL5321datasheet}.
% ADL5321datasheet Table 2 (S21 at 2850 and 2900 MHz, interpolated at 2877 MHz)
This stage is \textbf{not required for D0 CW-ODMR} and is deferred to D1 pulsed operations.

\subsection{D0 vs.\ D1 Readout Requirements}

For \textbf{D0} (Meissner--Zeeman POC): no photon-counting electronics are required.
CW-ODMR using the existing ADF4351 synthesizer and a lock-in or software sweep is sufficient.
The SiPM readout chain is designed for D1 (energy-resolving bolometry).

For \textbf{D1}: the full chain (Table~\ref{tab:readout_components}) must be assembled
and characterized at cryogenic temperature before use.
